RTL

Support for right-to-left text in Pico.

To enable RTL in Pico you need to set dir="rtl"on the <html> element (example).

<!doctype html>
<html dir="rtl" lang="ar">
  ...
</html>

ℹ️ The RTL feature is still experimental and will probably evolve.
